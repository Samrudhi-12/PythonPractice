# Python Practice

This repository contains beginner-level Python programs created for learning and practice purposes. The programs cover fundamental programming concepts such as variables, loops, conditional statements, functions, and basic problem solving.

## Files Included

- **hello.py** – Basic Hello World program  
- **learn.py** – Simple Python learning examples  
- **prime_number_checker.py** – Program to check whether a number is prime  
- **tic-tac-toe.py** – Console-based Tic Tac Toe game implemented in Python  

## Purpose

This repository is part of my programming practice and preparation for advanced studies in Artificial Intelligence.

## How to Run

1. Open the file in PyCharm or any Python IDE  
2. Run the program using Python 3
